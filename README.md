# 🎬 Movie Recommendation System

A content-based movie recommendation system built using Machine Learning techniques like TF-IDF and Cosine Similarity.

---

## 🚀 Project Overview

This project recommends movies based on their genres.  
If a user inputs a movie name, the system suggests similar movies.

Example:
> If you like *Toy Story*, you’ll get similar animated/adventure movies.

---

## 🧠 How It Works

1. Movie genres are converted into text format  
2. TF-IDF Vectorization transforms text into numerical vectors  
3. Cosine Similarity is used to measure similarity between movies  
4. Top similar movies are recommended  

---

## 🛠️ Tech Stack

- Python 🐍  
- Pandas  
- Scikit-learn  
- Streamlit (for UI)

---

## 📂 Dataset

- MovieLens Small Dataset  
- Contains movie titles and genres  

---

## 🎨 Features

- Search movies using partial names  
- Top 10 recommendations  
- Similarity score display  
- Interactive UI with Streamlit
